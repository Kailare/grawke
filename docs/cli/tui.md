# `grawke tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/tui)

## Examples

```bash
grawke tui
grawke tui --url ws://127.0.0.1:18789 --token <token>
grawke tui --session main --deliver
```
