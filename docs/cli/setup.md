# `moltx setup`

Initialize `~/.moltx/moltx.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
moltx setup
moltx setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
moltx setup --wizard
```
