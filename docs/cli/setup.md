# `grawke setup`

Initialize `~/.grawke/grawke.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
grawke setup
grawke setup --workspace ~/clawd
```

To run the wizard via setup:

```bash
grawke setup --wizard
```
